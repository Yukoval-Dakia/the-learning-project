数据流（与生产代码完全一致）
  1. GET /api/knowledge → nodes（含 mastery, evidence_count, domain, parent_id）
  2. GET /api/knowledge/edges → edges → 过滤 archived_at===null
  3. GET /api/knowledge/review-due-summary → dueCounts Map
  4. passesFilter（domain + mastery band + dueOnly）→ universeNodes
  5. progressive disclosure：roots 自动展开，点击节点 drill-down
  6. computeLayout（nodes ≤ 60 → tidyTree；> 60 → fcose headless）
    Harness 只实现 tidyTree；fcose 路径显示提示
  7. SVG 渲染：nodes + tree edges + mesh edges
快捷键
  • 拖拽 = 平移
  • Ctrl/Cmd + 滚轮 = 缩放
  • 双指滚动 = 平移
  • 点击节点 = 展开 / 收起 + 查看详情
  • 点击空白 = 退出聚焦
masteryBand 规则
evidence_count=0 → untrained，<3 → insufficient，mastery<0.4 → weak，<0.7 → learning，≥0.7 → mastered

masteryTone（碟片颜色）
mastery≥0.7 → good（绿），≥0.4 → hard（琥珀），其余 → again（红）
