# 文言文翻译题评分细则

本细则供两处共用：质检阶段 `kind_conformance` 检查（题目本身是否是一道合格翻译题）、以及未来学生答案的 semantic judge 判分（学生译文得几分）。

## 一、题目合格性细则（kind_conformance）

一道翻译题**合格**当且仅当：

1. 原文句是真实文言（非自造），长度 1–3 句、约 20–60 字。
2. 设了 2–4 个明确采分点，且至少覆盖「实词」「虚词功能」「句式」「活用」中的两类。
3. `reference_md` 给了完整、准确、通顺的现代汉语参考译文。
4. `rubric_json.required_points` 与采分点逐一对应、可核查。
5. `judge_kind_override='semantic'`。

任一不满足 → 不合格（kind_conformance 倾向 fail / unclear）。

## 二、学生译文判分细则（semantic judge，满分按采分点数折算）

- 每个采分点独立计分：踩中（关键字词译法正确 + 句式还原正确）得该点分，否则不得分。
- **整体通顺**作为一个隐含约束：即使采分点全中，若整句译文语义明显错乱 / 漏译主干，要扣整体分。
- 常见扣分：把古今异义词按今义直译；句式未还原（如被动句译成主动）；活用未译出（如使动当一般动词译）；漏译虚词导致语法关系丢失。
- 通假字、活用允许学生用不同表述，只要**语义等价**即判踩中（semantic 判分的容错点）。
