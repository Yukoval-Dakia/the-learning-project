# Triage Label Configuration

This project uses the standard triage label vocabulary.

## Label Mapping

| Role | Label |
|------|-------|
| `needs-triage` | `needs-triage` |
| `needs-info` | `needs-info` |
| `ready-for-agent` | `ready-for-agent` |
| `ready-for-human` | `ready-for-human` |
| `wontfix` | `wontfix` |

## Applying Labels

When the `triage` skill processes issues, it will apply these labels to move issues through the triage state machine.
