# Issue Tracker Configuration

This project uses **GitHub Issues** as its issue tracker.

## Workflow

Issues are created and managed using the GitHub CLI (`gh`).

## Creating Issues

Use `gh issue create` to create new issues in this repository.

## Reading Issues

Skills read issues using `gh issue view` and `gh issue list`.
