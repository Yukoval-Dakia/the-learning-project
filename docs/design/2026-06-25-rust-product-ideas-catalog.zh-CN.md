## 1. 即时与离线（WASM）——「在飞机上它就是能用」

1. **口袋练习（完整离线闭环）**——无信号也能完成一次完整的判分会话，回到信号区时再同步。——[WASM-client]——亮点：飞行模式、30 道题，每道即时判分，下一张卡片已在设备端选好，一个「已排队同步」的小标在累加——应用从不转圈。——中
2. **零延迟客观判分**——手指还没离开按键，结论就已落地。——[WASM-client]——亮点：提交到「正确」之间没有任何卡顿；`1/2 m` ≡ `50 cm` 被零往返地接受——判分像计算器一样。——易
3. **离线分步检查草稿板**——输入一段多步推导，每一行随你书写变绿/变红。——[WASM-client]——亮点：给推理用的拼写检查器，闭环里没有服务器。——中
4. **单位/量纲健全性指示灯**——边输入答案边做实时量纲一致性检查。——[WASM-client]——亮点：`kg·m/s²` 亮绿、`kg·m/s` 报警——内置进 WASM 判分核心。——易
5. **离线「下一步该做什么」选择器**——下一道题，由实时掌握度 + 到期日选出，完全在客户端。——[WASM-client + determinism]——亮点：做完一张卡，下一张 *已经* 在屏幕上，且与服务端会派发的那道一致。——中
6. **私密模式（数据永不离开设备）**——一个开关，把每一次作答、错误、估计都只留在本机。——[WASM-client]——亮点：练完一整个主题，打开 Network 面板——*零*请求，而判分与调度照常工作。——中
7. **WASM 内的实时知识图布局**——离线以 60fps 平移/缩放上千节点的概念图。——[WASM-client]——亮点：今天会卡顿的图变得顺滑;确定性种子意味着每次加载看起来 *一模一样*，所以空间记忆有效。——中
8. **空间稳定的渐进式展开**——展开一个簇；新节点绕它滑入，其余部分原地不动。——[WASM-client]——亮点：地图感觉像一个「地方」，而非幻灯片——只对被触碰的子图做锚定式增量松弛。——中
9. **即时语料搜索面板**——按 `/`，离线地对每个 KC、笔记、题目、上传做模糊搜索，边打边排序。——[WASM-client]——亮点：覆盖你整个学习生涯的 Spotlight，在飞机上也满速。——中
10. **本地语义召回（浏览器内 HNSW）**——用模糊的字眼输入一个半记得的想法；对的节点按 *含义* 亮起。——[WASM-client]——亮点：embedding 在服务端预算，但 ANN 检索在 WASM 中跑、查询下载下来的向量 blob。——中
11. **图中「边拖边搜」**——平移时边输入边让匹配节点发光、邻居淡出。——[WASM-client]——亮点：过滤索引就活在渲染器旁边的 WASM 里；没有 debounce 的 API 调用，60fps。——易
12. **我是不是已经学过这个？（上传去重）**——丢进一个文档；在任何服务端 job 之前，徽标就显示概念重叠。——[WASM-client]——亮点：「3 个 KC 你已经有了，2 个是新的」会在 LLM 打标启动 *之前* 就出现。——中
13. **即时学习集构建器（离线分面搜索）**——通过对整个题库做「结构化 + 模糊」实时查询来组合一个练习集。——[WASM-client]——亮点：「本簇内的 KC、难度 > X、最近答错、30 天未见」这类过滤瞬间筛过上千条。——中
14. **本地自适应难度（实时 IRT 定位）**——下一道题的难度按实时 θ̂ 调到约 75% 成功率的甜点区，离线。——[WASM-client + determinism]——亮点：会话会「呼吸」——连对后变难、卡壳后变易——不联网。——中
15. **离线 CAT（完整自适应诊断）**——在飞机上做一整套自适应分级测验，每道题都选得恰到好处。——[WASM-client]——亮点：WASM 里的 Fisher 最大化选题器跑完整个自适应闭环，重连后再同步。——中
16. **页面加载时的 WASM 掌握度预计算**——离线打开，你的整张掌握度档案已经画好了。——[WASM-client + determinism]——亮点：环形图在任何网络调用返回 *之前* 就从缓存的事件尾部渲染出来，随后无闪烁地对账。——中
17. **本地间隔重复提醒**——你的设备在数学上最优的那一刻提醒你，全程端上计算。——[WASM-client]——亮点：离线且私密的正确提醒——没有任何服务器需要知道你的日程。——易

## 2. 模拟与预测（native-compute）——「把我的大脑向前跑 10,000 次」

18. **学习者的数字孪生**——前向模拟你的知识，告诉你 30 天后实际会到哪。——[native-compute]——亮点：每 KC 一张 *扇形图*——从 θ̂/SE 出发对 FSRS×PFA 做蒙特卡洛采样前向推演，给出预测掌握度中位数 + 10–90% 误差带。——中
19. **What-If 时间滑块**——把每天 10→90 分钟拖来拖去，实时看预测曲线 *呼吸*。——[native-compute]——亮点：拖动期间 *每一帧* 都重新模拟上千种未来。——中
20. **最优日程搜索（蒙特卡洛规划器）**——「我在 0.4 秒里模拟了 5,000 种日程——这一种比别的早 47 分钟达成你的目标。」——[native-compute]——亮点：枚举间隔，对每种做完整前向 MC，返回帕累托最优并显式给出模拟次数。——登月级
21. **目标日期反解器**——「想在 18 号前考试就绪？这是每日剂量——或者诚实地告诉你『最早是 23 号』。」——[native-compute]——亮点：在每日投入量上搜索，每次探测都是一整张图的 MC。——中
22. **先修级联模拟器**——让某个地基烂掉，看损伤沿图向上扩散。——[native-compute]——亮点：在模拟里删掉一个 KC，沿先修边在上千条轨迹上传播；下游节点按概率加权的严重度变暗。——中
23. **策略锦标赛（A/B/C 计划）**——突击 vs 稳扎 vs 交错——三种未来并排。——[native-compute]——亮点：在命名策略下跑孪生，扇形图并排，附峰值/第 60 天/总分钟数/倦怠代理指标。——中
24. **后悔计（跳过代价预测）**——「今天跳过，−4%；跳过三天，−14%。」——[WASM-client]——亮点：在收尾时于浏览器内重算「执行计划 vs 啥也不做」的反事实 MC，离线也行。——易
25. **遗忘悬崖雷达（分带）**——「这 7 个概念会在你下次见到它们之前崩掉。」——[native-compute]——亮点：对每个 KC 做可提取性穿越的 MC，得到失败日期的 *分布*，并按承重权重排序。——易
26. **过载再平衡器**——「下周二会飙到 40 个复习——这是一份达成同样目标但更平滑的计划。」——[native-compute]——亮点：检测 FSRS 堆积，再做 MC 搜索一份负载均衡、且预测掌握度在统计上无法区分的日程。——中
27. **全图掌握度-ROI 图**——每个节点显示「到扎实还要几次会话」；按「投入-回报」对整张图排序。——[native-compute]——亮点：对 *所有* KC 同时前向模拟 PFA+FSRS，每个节点带 ETA 误差带，可按 ROI 实时排序。——易
28. **新概念就绪门**——「你现在可以开始这个，但会很挣扎——先夯实这 2 个先修，上手会顺 3 倍。」——[native-compute]——亮点：在先修子图上做「现在就开始 vs 先夯实先修」的 MC。——中
29. **隔夜边际收益排序器（学习自动驾驶）**——「你睡觉时我跑了约 10,000 个版本的明天，并把最好的预烤好了。」——[native-compute]——亮点：早餐时呈上一份排好序的会话 + 「我考虑了 9,847 个计划；这个能提升你最弱的 3 个先修，又不会压垮复习债。」——中

31. **间隔重复风洞（参数工作室）**——拖动 FSRS 旋钮，看预测遗忘曲线在你整副牌上弯折。——[native-compute]——亮点：滑动保留率目标，上千张卡片即时重新模拟，绘出「复习负载 vs 保留率」。——中
32. **掌握度天气预报**——「3 个概念周四前转阴，下周若今天跳过会有一场暴风雨。」——[native-compute]——亮点：天气式地图，每个区域配预报图标 + 置信带。——中
33. **误解传染图**——看一个动摇的概念如何「感染」下游 KC——一场在你先修图上的 SIR 传染病。——[native-compute]——亮点：从弱节点播种的扩散动画展示哪些概念有崩塌风险。——中

## 3. 确定性回放与可信——「你的进度是可审计的，不是黑盒」

38. **回放刮擦条 / 时光机**——拖动时间轴，逐帧看你的整张知识图从空白长到今天。——[determinism]——亮点：刮到任意过去日期；浏览器把完整事件日志重投影到那一刻，零服务端调用——你能看到某个误解死去的确切那天。——登月级
39. **你思想的 Git-Diff**——挑两个日期；得到一份字面意义的变更日志：新增的 KC、获得的掌握度、重连的边、被划掉的误解。——[determinism]——亮点：读起来像在 review 一个针对你自己的 pull request。——中

## 诚实的边界——Rust 并 *不* 解锁什么

Rust 恰好买到三样东西：**CPU 密集型数值计算**（IRT/EM/MCMC/MIRT/Fisher/蒙特卡洛/聚类/等值）、**WASM 端上 + 离线** 执行、以及 **位级精确确定性**（服务端≡客户端、可复现）。在「I/O- 与模型受限」的那一半上它买到 **零**：AI 家教的散文/提示/解释；视觉/OCR 录入与手写判分；开放式/主观判分；LLM 概念打标与边提议；copilot 对话；*生成* embedding；普通的 DB CRUD——全都被模型端点或 Postgres 的延迟所主导，Rust 在那里加成约等于 0。有几个点子（本地去重 #117、易混概念 #120、重排 #119）声称 Rust 价值 *仅* 在于对 **已存在** 的向量做相似度/聚类数学——创造这些向量的 embedding 步骤是一次 LLM 调用，超出范围。经验法则：如果惊艳点是「AI 说了句聪明话」，那不是 Rust 特性；如果惊艳点是「这个数字是即时的、离线的、或可证明正确的」，那就是。
