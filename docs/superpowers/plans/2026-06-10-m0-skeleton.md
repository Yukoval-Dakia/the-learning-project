# M0：新骨架示踪弹 — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans。Steps 用 checkbox 跟踪。

**Goal:** 一发示踪弹穿透全新栈：浏览器 → Vite SPA（TanStack Router）→ Hono（token gate）→ agent-notes GET（manifest→组合根**真实挂载**，零壳文件）→ Postgres。

**Architecture:** 总 spec REV 2 §2.5/§4-M0（D17-D20）。原仓拆旧建新：新根 `server/`（Hono）与 `web/`（Vite SPA）；旧 app/ 是采石场，本里程碑执行第一次合法拆除（旧 agent-notes 壳页）。连接既有 dev Postgres（schema 新基线推迟 M1）。

**Linear:** YUK-313。分支 `yuk-313-m0-skeleton`。

**关键设计决策（执行者必读）：**
1. **handler 懒加载**：manifest 的 route 声明带 `load: () => import('../api/notes').then(m => m.GET)`——静态元数据保持纯净（composition.unit.test 不会拉进 @/db/client），server 挂载时才解析。param 路由的 handler 适配推迟到 M1 首个 `[id]` 路由（第二实例原则）；M0 仅 `load` 可选字段 + 无参 handler 契约 `(req: Request) => Promise<Response>`。
2. **env 自加载**：`server/env.ts` 零依赖 fs 解析 .env / .env.local（已设的不覆盖），不赌 tsx 的 --env-file 兼容性。
3. **拆除而非适配**：`src/capabilities/agent-notes/ui/page.tsx` 的 next/navigation 直接换成 TanStack Router 等价物，同 commit 删除旧壳 `app/(app)/agent-notes/page.tsx`（D18：新 surface 上线即删旧 surface；today 页的看全部链接 404 属预期，M4 工作台重接）。
4. **样式复用**：web 直接 `import '../../app/globals.css'`（不复制不搬家——避免双真相源；M5 拆除时才迁移归属）+ `@tailwindcss/vite` 编译 utilities。

- [ ] `server/app.ts`——`buildHonoApp(capabilities: CapabilityManifest[]): Hono`：

```ts
import { Hono } from 'hono';
import type { CapabilityManifest } from '@/kernel/manifest';

/** Next 风格 '[id]' 段 → Hono ':id'（M0 无参路由用不到，先留转换器 + 单测） */
export function toHonoPath(path: string): string {
  return path.replace(/\[([^\]]+)\]/g, ':$1');
}

export function buildHonoApp(capabilities: CapabilityManifest[]): Hono {
  const app = new Hono();
  app.use('/api/*', async (c, next) => {
    if (c.req.path === '/api/health') return next();
    if (c.req.header('x-internal-token') !== process.env.INTERNAL_TOKEN) {
      return c.json({ error: 'unauthorized' }, 401);
    }
    return next();
  });
  app.get('/api/health', (c) => c.json({ ok: true }));
  for (const cap of capabilities) {
    for (const route of cap.api?.routes ?? []) {
      if (!route.load) continue;
      const load = route.load;
      app.on(route.method, toHonoPath(route.path), async (c) => (await load())(c.req.raw));
    }
  }
  return app;
}
```
