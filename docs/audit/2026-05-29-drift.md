#### [ADR-0022 §Save Contract / ADR-0011] `experimental:artifact_section_edit` — 未 catalog 的 experimental action

- **声明**: ADR-0022 §Save Contract: "writes `experimental:artifact_body_blocks_edit`"（全文档保存事件）。ADR-0011 记录 experimental 命名空间 action 列表；`RESERVED_EXPERIMENTAL_ACTIONS`（`src/core/schema/event/experimental.ts`）当前仅含 `experimental:user_cause` / `experimental:record_capture` / `experimental:memory_brief_refresh`。
- **代码**: `src/server/artifacts/sections.ts:153` — `writeEvent(..., action: 'experimental:artifact_section_edit', ...)` 于 section-level PATCH 路径写入。`src/server/artifacts/sections.test.ts:114` 验证该 action name。对应 API 路由 `app/api/artifacts/[id]/sections/[sectionId]/route.ts` 仍在生产中供 YUK-54 section-edit 流使用。
- **冲突**: `experimental:artifact_section_edit` 既不在 `RESERVED_EXPERIMENTAL_ACTIONS`（因此 payload 松散、无专属 schema 守卫），也不在任何 ADR/save-contract 文档中注明 payload shape 或 stabilization criteria。ADR-0022 只文档了全文档保存路径的 mirror event，缺少 section-patch 路径的对应条目。
- **建议**: 有两条路：(a) 在 ADR-0022 §Save Contract 补注 section-patch event shape，并加入 `RESERVED_EXPERIMENTAL_ACTIONS` + 专属 schema（推荐，可与 ADR-0011 promotion sweep 一并处理）；或 (b) ADR-0020 已约定编辑应走 block_id 粒度，若 section-patch 路径在 Living Note v0（Wave 6 T-88 P4）后被 TipTap 全文档 save 取代，可在 P4-E closeout 时 deprecate 此端点并移除 event。
- Linear issue（closeout gate 去重后创建）：
  - title: `Drift: ADR-0022 ↔ experimental:artifact_section_edit undocumented`
  - labels: `drift`

#### [ADR-0017 §Write triggers #2] Chat-derived Mem0 add path 未实现

- **声明**: ADR-0017 §Write triggers: "**2. Chat-derived** — Copilot conversation turns can produce user-preference facts... orchestrator decides whether to call `mem0.add(chat_message, scope='meta:orchestrator_self')`"。
- **代码**: `src/server/memory/triggers.ts` 仅实现 trigger #1（outbox poller → `client.addEventMemory`）和 trigger #3（cron sweep via pg-boss）。`createMemoryClient()` / `memoryClient.add` 在任何 Copilot/teaching-session 路径（`app/api/teaching-sessions/route.ts` / `src/server/copilot/` / `src/server/orchestrator/`）中均无调用。`meta:orchestrator_self` scope 已在 `brief.ts` 模板注册（`src/server/memory/brief.ts:15` / `scope_tagger.ts:75`），但没有任何 Copilot turn 触发 `mem0.add(chat, scope='meta:orchestrator_self')`。
