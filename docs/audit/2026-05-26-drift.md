- **声明**: ADR-0014 §核心概念："框架本身不懂任何科目"。Foundation B M1 closeout（commit `6a69431`）明确"Neutralize 3 deprecated registry fallback strings"，math M1 drift doc（`docs/superpowers/plans/2026-05-22-math-m1-drift-targets.md §A`）列出 4 个 task，Action(M3) 要求全部替换为 placeholder 或移除。

- **冲突**: Foundation B M1 已完成 3/4（NoteGenerateTask / VariantGenTask / TeachingTurnTask 均替为 `'(see getTaskSystemPrompt(task, profile) - fallback not for runtime)'`），但 KnowledgeReviewTask 仅添加了 DEPRECATED 注释，字符串内容未替换。runtime 走 `buildKnowledgeReviewPrompt(profile)`（`task-prompts.ts:318`），不读 registry 字段，无运行时影响；但该字符串对代码阅读者造成误导——让人以为图谱维护助手对新 subject 仍有 wenyan-only 限制。
- **建议**: 把 KnowledgeReviewTask.systemPrompt 替换为 `'(see getTaskSystemPrompt(task, profile) - fallback not for runtime)'`，与其他 3 个 task 保持一致。单行改动，无测试变更。
